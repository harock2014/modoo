기본 그래프 그리기¶

파이썬 데이터 시각화 라이브러리 matplotlib 를 사용하여 그래프를 그린다.

matplotlib 홈페이지 : https://matplotlib.org

In [1]:
import matplotlib.pyplot as plt   # matplotlib의 pyplot를 plt로 불러옴.

plt.plot([10, 20, 30, 40])

plt.show()

<Figure size 640x480 with 1 Axes>

x 값을 명시적으로 입력하지 않을 경우 0, 1, 2 순으로 자동으로 부여됨. plot 함수의 인자값이 하나일 경우 입력된 값을 y 값으로 인식함.

따라서 위 그래프와 같이 (0,10), (1,20), (3,30), (4,40) 의 형태로 나타남. 그래프 역시 선형 그래프로 표시됨.

In [2]:
plt.plot([1,2,3,4], [10, 20, 30, 40])   #plt.plot([X축 테이터],[Y축 데이터]) 

plt.show()

그래프에 범례 넣기¶

In [4]:
import matplotlib.pyplot as plt
plt.title('legend')
plt.plot([10, 20, 30, 40], label ='asc') # 증가를 의미하는 asc 범례
plt.plot([40, 30, 20, 10], label ='desc') # 감소를 의미하는 desc 범례
plt.plot([20, 20, 20, 20], label ='y=20') # 감소를 의미하는 desc 범례
plt.legend()  # legend함수를 각 plot 함수의 label을 모아 범례로 표시함.
plt.show()

import matplotlib.pyplot as plt
plt.title('legend')
plt.plot([10, 20, 30, 40], label ='asc') # 증가를 의미하는 asc 범례
plt.plot([40, 30, 20, 10], label ='desc') # 감소를 의미하는 desc 범례
plt.plot([20, 20, 20, 20], label ='y=20') # 감소를 의미하는 desc 범례
plt.legend(loc=2)  # legend함수를 각 plot 함수의 label을 모아 범례로 표시함.
plt.show()

범례 표시 위치 조정 2 : 좌상단 9 : 상단중앙 1 : 우상단 6 : 좌중단 10 : 중앙 5 : 우중단 7 : 우중단 3 : 좌하단 8 : 좌중단 4 : 우하단

그래프 색상 바꾸기¶

In [6]:
import matplotlib.pyplot as plt
plt.title('color') # 제목 설정
plt.plot([10, 20, 30, 40], color ='skyblue', label ='skyblue') # 하늘색 그래프
plt.plot([40, 30, 20, 10], 'pink', label ='pink') # 분홍색 그래프
plt.legend() # 범례 표시
plt.show()

그래프 선 모양 바꾸기¶

In [7]:
import matplotlib.pyplot as plt
plt.title('linestyle') #제목 설정
plt.plot([10, 20, 30, 40], color ='r', linestyle ='--', label
='dashed') # 빨간색 dashed 그래프
plt.plot([40, 30, 20, 10], color ='g', ls =':', label ='dotted')
# 초록색 dotted 그래프
plt.legend() # 범례 표시
plt.show()

마커 모양 바꾸기¶

In [8]:
import matplotlib.pyplot as plt
plt.title('marker') # 제목 설정
plt.plot([10, 20, 30, 40], 'r.', label ='circle')   # 빨간색 원형 마커 그래프
plt.plot([40, 30, 20, 10], 'g^', label ='triangle up')   # 초록색 삼각형 마커 그래프
plt.legend() # 범례 표시
plt.show()
